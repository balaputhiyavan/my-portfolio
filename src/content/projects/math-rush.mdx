## Overview
**MathRush** is an engaging puzzle game designed to sharpen arithmetic skills under pressure. Developed entirely in **Flutter**, it demonstrates the framework's capability for 2D game logic and smooth animations. The project encompasses the full mobile lifecycle, from game loop implementation to monetization integration with **Google AdMob**.

### Core Features
1.  **Gameplay Loop**: Rapid-fire question generation with increasing difficulty curves.
2.  **AdMob Integration**: Banner ads and Rewarded Video ads for retries/power-ups.
3.  **State Management**: Optimized performance using Riverpod for game state and scoring.
4.  **Animations**: Custom Rive animations for correct/wrong answers and level transitions.

### Tech Stack
-   **Flutter**: UI and Game Logic.
-   **Google AdMob**: Monetization SDK.
-   **Shared Preferences**: For local high-score persistence.
