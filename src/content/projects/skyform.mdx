## Overview
**SkyForm** is a native iOS application engineered for reliability and scalability. It serves as a comprehensive tool for collecting survey data in the field. By adopting strict **Clean Architecture** principles and **MVVM**, the codebase is testable, modular, and easy to maintain. It showcases advanced usage of **SwiftUI** for declarative UI and **Core Data** for complex object graph persistence.

### Core Features
1.  **Offline-First**: All data is stored locally in Core Data and synced when online.
2.  **Dynamic Forms**: UI renders based on JSON schema configurations for flexible survey types.
3.  **Clean Architecture**: Separation of concerns into Domain, Data, and Presentation layers.
4.  **Reactive Bindings**: Extensive use of `Combine` publishers for real-time validation.

### Tech Stack
-   **SwiftUI**: Modern declarative UI.

-   **Combine**: Reactive functional programming.
